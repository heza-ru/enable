# Whatfix Master Deck Reference

This folder contains your company's master presentation deck templates that the AI will use as reference when generating presentations.

## How to Use

1. **Add your master deck content** to `master-deck-structure.md`
2. **Customize slide templates** in `slide-templates.md`
3. **Update icons and visuals** in `visual-guidelines.md`
4. The AI will automatically use these as reference when generating presentations

## Files in This Folder

- **`master-deck-structure.md`** - Your standard deck structure and slide types
- **`slide-templates.md`** - Reusable slide templates with examples
- **`visual-guidelines.md`** - Icons, colors, fonts, and visual standards
- **`messaging-framework.md`** - Key messaging, value props, and terminology

## Quick Start

If you have an existing PowerPoint master deck:

1. Extract the slide types and structure
2. Copy the text content from each slide type
3. Note any icons, colors, or visual elements used
4. Paste everything into the respective template files

The AI will learn your company's presentation style and generate on-brand decks automatically!

## Example

Instead of asking: "Create a sales deck"

The AI will generate slides matching your:
- Corporate slide structure
- Standard messaging
- Visual guidelines
- Approved terminology
