## üí° Pro Tips

### Start Small
Don't feel pressured to fill out everything at once. Start with:
1. 3-5 most common slide types
2. Your core messaging
3. Your brand colors

### Use Real Examples
Copy actual text from your best presentations. Real examples work better than descriptions.

### Be Specific
Instead of: "Use corporate colors"
Write: "Primary: #0066FF, Secondary: #00CC88"

### Include Context
Add "when to use" guidance for each template.

### Update Regularly  
As your messaging evolves, update the templates. Treat this as a living document.

## üÜò Need Help?

### If Templates Aren't Working
- Check for typos in markdown formatting
- Ensure you removed all `[Add your...]` placeholders
- Verify files are saved in UTF-8 encoding

### If AI Ignores Templates
- Make sure templates are detailed and specific
- Add more examples
- Use actual text from your decks, not descriptions

### If Output Quality Is Low
- Provide more context in your request
- Set customer context before generating
- Iterate with specific feedback
